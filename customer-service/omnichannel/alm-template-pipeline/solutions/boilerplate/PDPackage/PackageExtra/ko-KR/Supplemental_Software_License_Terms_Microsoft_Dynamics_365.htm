참고: Microsoft Corporation(또는 거주 지역에 따라 계열사 중 하나)은 귀하에게 본 추가 구성 요소의 사용권을 허여합니다. 귀하는 각각 유효하게 사용권이 허여된 (이 추가 구성 요소가 해당되는) Microsoft Dynamics 365소프트웨어("소프트웨어")의 사본 또는 각각 유효하게 사용권이 허여된 Microsoft Dynamics 365 서비스의 정기가입("온라인 서비스")와 함께 이 추가 구성 요소의 사본을 사용할 수 있습니다. 해당 소프트웨어 또는 온라인 서비스에 대한 사용권이 없는 경우에는 추가 구성 요소도 사용할 수 없습니다. 추가 구성 요소 사용 시 소프트웨어에 대한 사용권 계약 또는 서비스가 적용됩니다. Microsoft는 https://support.microsoft.com/common/international.aspx에서 설명된 바와 같이 추가 구성 요소에 대한 지원 서비스를 제공합니다.
