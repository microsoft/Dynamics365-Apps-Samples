โปรดทราบว่า: Microsoft Corporation (หรือหนึ่งในบริษัทในเครือในประเทศที่ท่านอาศัยอยู่) ได้ให้สิทธิการใช้งานส่วนประกอบเพิ่มเติมนี้แก่ท่าน ท่านอาจใช้ผลิตภัณฑ์นี้กับสำเนาที่ได้รับอนุญาตอย่างถูกต้องของซอฟต์แวร์ Microsoft Dynamics CRM 365 (ที่ส่วนประกอบเพิ่มเติมนี้สามารถใช้ได้) (“ซอฟต์แวร์”) หรือการสมัครเป็นสมาชิกที่ได้รับอนุญาตอย่างถูกต้องของบริการ Microsoft Dynamics 365 (“Online Service”) ท่านไม่สามารถใช้ส่วนประกอบเพิ่มเติม หากท่านไม่มีสิทธิการใช้งานสำหรับซอฟต์แวร์หรือ Online Service เงื่อนไขสิทธิการใช้งานสำหรับซอฟต์แวร์หรือบริการจะมีผลใช้กับการใช้งานส่วนประกอบเพิ่มเติมนี้ของท่าน Microsoft ให้บริการสนับสนุนสำหรับส่วนประกอบเพิ่มเติมนี้ดังที่อธิบายไว้ใน https://support.microsoft.com/common/international.aspx
